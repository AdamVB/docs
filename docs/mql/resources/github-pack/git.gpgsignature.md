# git.gpgSignature

**Description**

Git GPG signature

**Fields**

| ID        | TYPE   | DESCRIPTION                       |
| --------- | ------ | --------------------------------- |
| sha       | string | GPG signature shasum              |
| reason    | string | GPG signature reason              |
| verified  | bool   | Whether GPG signature is verified |
| payload   | string | GPG signature payload             |
| signature | string | GPG signature                     |
