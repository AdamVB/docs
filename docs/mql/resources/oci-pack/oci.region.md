# oci.region

**Description**

Oracle Cloud Infrastructure (OCI) region

**Fields**

| ID           | TYPE   | DESCRIPTION                           |
| ------------ | ------ | ------------------------------------- |
| id           | string | Region's key                          |
| name         | string | Region name                           |
| isHomeRegion | bool   | Whether the region is the home region |
| status       | string | Region subscription status            |
