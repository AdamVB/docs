# oci.identity.apiKey

**Description**

Oracle Cloud Infrastructure (OCI) API key (a PEM-format RSA credential)

**Fields**

| ID          | TYPE   | DESCRIPTION                   |
| ----------- | ------ | ----------------------------- |
| id          | string | Key ID                        |
| value       | string | Key value                     |
| fingerprint | string | Key fingerprint               |
| created     | time   | Time when the key was created |
| state       | string | Key state                     |
