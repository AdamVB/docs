# esxi.timezone

**Supported platform**

- vmware-esxi
- vmware-vsphere

**Description**

vSphere ESXi timezone resource

**Fields**

| ID          | TYPE   | DESCRIPTION                  |
| ----------- | ------ | ---------------------------- |
| key         | string | Identifier for the time zone |
| name        | string | Time zone name               |
| description | string | Description of the time zone |
| offset      | int    | GMT offset                   |
