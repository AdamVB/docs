# vsphere.vmnic

**Supported platform**

- vmware-esxi
- vmware-vsphere

**Description**

vSphere ESXi physical network interface resource

**Fields**

| ID          | TYPE   | DESCRIPTION                               |
| ----------- | ------ | ----------------------------------------- |
| name        | string | NIC name                                  |
| properties  | dict   | NIC properties                            |
| details     | dict   | Generic configuration of a network device |
| pauseParams | dict   | NIC pause parameters                      |
