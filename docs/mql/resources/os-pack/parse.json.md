# parse.json

**Description**

Parse JSON files

**Init**

parse.json(path string)

**Fields**

| ID      | TYPE            | DESCRIPTION                                |
| ------- | --------------- | ------------------------------------------ |
| file    | [file](file.md) | File that is parsed                        |
| content | string          | Raw content of the file that is parsed     |
| params  | dict            | The parsed parameters defined in this file |
