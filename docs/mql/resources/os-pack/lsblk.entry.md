# lsblk.entry

**Description**

Unix block device

**Fields**

| ID          | TYPE             | DESCRIPTION                 |
| ----------- | ---------------- | --------------------------- |
| name        | string           | Device name                 |
| fstype      | string           | File system type            |
| label       | string           | Label for the file system   |
| uuid        | string           | UUID for the file system    |
| mountpoints | &#91;&#93;string | Mount points for the device |
