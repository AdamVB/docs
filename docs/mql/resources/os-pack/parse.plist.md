# parse.plist

**Description**

Parse plist files

**Init**

parse.plist(path string)

**Fields**

| ID      | TYPE            | DESCRIPTION                                         |
| ------- | --------------- | --------------------------------------------------- |
| file    | [file](file.md) | File that is parsed                                 |
| content | string          | Raw content of the file that is parsed              |
| params  | dict            | The parsed parameters that are defined in this file |
