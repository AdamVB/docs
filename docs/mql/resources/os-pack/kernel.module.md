# kernel.module

**Description**

System kernel module information

**Init**

kernel.module(name string)

**Fields**

| ID     | TYPE   | DESCRIPTION                  |
| ------ | ------ | ---------------------------- |
| name   | string | Name of the kernel module    |
| size   | string | Size of the kernel module    |
| loaded | bool   | Whether the module is loaded |
