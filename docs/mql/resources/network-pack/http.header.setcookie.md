# http.header.setCookie

**Maturity**

experimental

**Description**

Set-Cookie header

**Fields**

| ID     | TYPE              | DESCRIPTION                                   |
| ------ | ----------------- | --------------------------------------------- |
| name   | string            | Name of the cookie to set                     |
| value  | string            | Value of the cookie to set                    |
| params | map[string]string | Additional parameters for setting this cookie |
